# Real-Time Sports Analytics Dashboard

A dynamic system that fetches and visualizes live match data with real-time updates, team logos, and rankings using GOOGLE SHEETS.

# Features
- Real-time score updates without page refresh
- Dynamic team ranking system
- Logo integration for visual clarity
- Responsive UI for different screen sizes
- Modular structure for easy scaling

# Languages Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (if used)
- Data Handling: Fetch API / WebSockets
